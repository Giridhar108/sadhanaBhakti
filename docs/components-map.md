# Components map

## App

- `src/app/App.tsx` — switches between dashboard and components preview via query string.

## Pages

- `src/pages/DashboardPage/DashboardPage.tsx` — assembled full dashboard.
- `src/pages/ComponentsPreviewPage/ComponentsPreviewPage.tsx` — isolated preview of individual components.

## Widgets

- `Sidebar` — left navigation and goal reminder.
- `Header` — title, search, notifications and profile.
- `MemorizationSection` — top verse cards carousel-like section.
- `ProgressChartCard` — visual progress chart.
- `PracticeCardsGrid` — grid containing Japa, Reading, Verses cards.
- `PracticeCard` — reusable card with progress ring and stats.
- `QuoteCard` — quote area with Krishna image and decorative lotus.
- `CalendarCard` — right-side calendar and streak card.
- `FriendsCard` — friends list.
- `RemindersCard` — reminders list with switches.

## Shared UI

- `Card` — base card wrapper.
- `Icon` — inline SVG icons.
- `ProgressRing` — circular progress.
