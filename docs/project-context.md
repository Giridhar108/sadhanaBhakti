# Project context

Проект: веб-приложение «Садхана Бхакти» / «Харе Кришна» для трекинга духовной практики.

Основные практики:

- джапа;
- чтение книг;
- изучение и запоминание стихов;
- календарь практики;
- друзья и поддержка;
- напоминания;
- общий прогресс.

Цель dashboard: дать пользователю ощущение спокойного личного пространства, где видно движение в практике без давления и перегруза.

## Текущий визуальный курс

- мягкий пастельный UI;
- молочный фон;
- светлые карточки;
- тонкие рамки;
- лавандовые, зелёные и золотые акценты;
- закруглённые блоки;
- мягкие тени;
- изображения лотоса, Кришны, аватаров и декоративных элементов.

## Текущая рабочая стратегия

Не работать больше с одним большим HTML как с монолитом. Проект разбит на React-компоненты, чтобы можно было безопасно править отдельные блоки и потом собирать их обратно в общий dashboard.
